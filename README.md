# Django_project
1) pip install Django
2) django-admin startproject proje_adı
3) proje_adı/
├── proje_adı/
│   ├── __init__.py
│   ├── settings.py
│   ├── urls.py
│   └── wsgi.py
├── manage.py
└── venv/ (isteğe bağlı - sanal ortam klasörü)

4) python manage.py runserver
5) python manage.py startapp uygulama_adı
